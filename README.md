# ClickSpace - Final Tanpa JavaScript

Project ini menggunakan HTML dan CSS saja.

## Alur Website
1. index.html
2. register.html
3. login.html
4. home-login.html
5. booking-form.html
6. success.html
7. status.html

## Catatan Alur
Karena tidak memakai JavaScript atau database, riwayat tidak bisa tersimpan otomatis.
Untuk membuat alur tetap jelas, dibuat dua kondisi:
- status-empty.html = kondisi sudah login tetapi belum booking
- status.html = kondisi setelah booking berhasil

Setelah booking berhasil, halaman success.html mengarahkan user ke status.html.
